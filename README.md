# Igala Dataset Explorer

A Streamlit web app for exploring an Igala text dataset.

## Features
- Dataset preview
- Sentence length statistics
- Top 20 most common words
- Bigram extraction
- Interactive filters

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
